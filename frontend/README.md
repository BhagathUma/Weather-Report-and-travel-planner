# Atmos AI

AI-Powered Weather Intelligence Platform

Transform weather forecasts into actionable recommendations using AI.

---

## Features

- AI-powered weather insights
- Personalized recommendations
- Travel planning assistance
- Outdoor activity forecasting
- Sports weather analysis
- Group event planning
- MongoDB intelligent caching
- Analysis history
- Premium glassmorphic UI

---

## Tech Stack

### Frontend

- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand

### Backend

- FastAPI
- Python
- Motor

### AI

- gemini-2.5-flash-lite

### Database

- MongoDB Atlas

### APIs

- Open-Meteo Forecast API
- Open-Meteo Geocoding API
